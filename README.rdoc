Pauta prueba final DesafioLatam
*******************************

Un admin sube productos, esos productos tienen una duración de dos horas.
los usuarios pueden ofertar para adquirir esos productos, al ofertar el tiempo de duración del producto aumenta en 2 minutos.

Cada oferta aumenta en uno el precio del producto, pasado el tiempo el último usuario que ofertó gana el producto.

Un usuario no puede ofertar dos veces consecutivas sobre el mismo producto.

Para probar el programa puedes correr los seeds que agregan productos a la base de datos con rake db:seeds

luego tienes que hacer tu propia cuenta y empezar a ofertar.

El panel de administrador tiene las claves por defecto
